# Support

Thanks for using CareCompanion. Here's the fastest way to get help.

## 📖 Start with the docs

- **[README](README.md)** — features, quick start, deployment
- **[Architecture](ARCHITECTURE.md)** — how the app is built and where data lives
- **[Contributing](CONTRIBUTING.md)** — local setup and conventions
- **[Security & Privacy](SECURITY.md)** — the full data-handling policy

## 🐞 Found a bug or want a feature?

Open an issue — the templates guide you through the details we need:

- [Bug report](../../issues/new?template=bug_report.yml)
- [Feature request](../../issues/new?template=feature_request.yml)

Before filing, a quick search of [existing issues](../../issues) often finds an
answer or an open thread you can add to.

## 💬 Questions & ideas

For open-ended questions, use [GitHub Discussions](../../discussions) if enabled,
otherwise open a regular issue.

## 🔒 Security issues

**Do not open a public issue for a vulnerability.** Follow the private disclosure
process in [SECURITY.md](SECURITY.md).

## 🩺 A note on medical advice

CareCompanion is a caregiving organiser, **not a medical device**. It does not
provide medical advice. For health decisions, always consult a qualified
professional.
